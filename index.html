Getting Started with RWpreise64

RWpreise64 is Vagrant box for rails developers. Vagrant uses Oracleâ€™s VirtualBox to build configurable, lightweight, and portable virtual machines dynamically.

Next install vagrant via RubyGems: 

gem install vagrant

RWpreise64 contains:

  • RVM
  • Ruby: Ruby 1.9.2 & 1.9.3, JRuby 1.6, Rubinius
  • MySql
  • PostgreSQL
  • Redis
  • MongoDB
  • Dev libs for compiled gems

Usage

Basic setup:

vagrant box add rwpreise64 http://dl.dropbox.com/u/83704794/vagrant/rwpreise64.box
vagrant init rwpreise64
vagrant up

After running vagrant ssh, you should see something similar to the following: 

$ vagrant ssh
...
vagrant@vagrantup:~$

Suspending the Environment: 

$ vagrant suspend

To resume working: 

$ vagrant resume

Halting the Environment: 

$ vagrant halt

Destroying the Environment: 

$ vagrant destroy
